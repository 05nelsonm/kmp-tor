FlagsBuilder

class FlagsBuilder(source)

Configure flags for the TorCmd.Onion.Add object, as described in control-spec#ADD_ONION.

NOTE: Flag V3Auth is not included here as an option. It is automatically added if clientAuth is configured.

Configurability is as follows:

  • null: no action (default).

  • true: add the flag if not present.

  • false: remove the flag if present.

e.g.

flags {
    Detach = true
    DiscardPK = true
}
flags {
    // Remove what was just added
    Detach = false
}

// ...

println(onionAddCmd.flags)
// [DiscardPK]
