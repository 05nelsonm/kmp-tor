startDaemonSync

@JvmStatic
@JvmOverloads
inline fun <T : Action.Processor> T.startDaemonSync(noinline cancellation: () -> CancellationException?? = null): T(source)

Starts the tor daemon, blocking the current thread until completion or cancellation/error.

NOTE: This is a blocking call and should be invoked from a background thread.

Parameters

cancellation

optional callback which is invoked after every thread sleep (so, multiple times) in order to trigger job cancellation if a non-null exception value is returned.
